# tm-user-js-mirror
Tampermonkey用戶腳本鏡像.  
對於本repo中的腳本, 直接訪問 `https://cdn.jsdelivr.net/gh/smalllqiang/tm-user-js-mirror@main/js/xxx.user.js` 即可在Tampermonkey中添加用戶腳本.  

## 使用的腳本
以下列出腳本url, 方便安裝.

- [Mediawiki IPE NEXT](https://cdn.jsdelivr.net/gh/smalllqiang/tm-user-js-mirror@main/js/Mediawiki_IPE_NEXT.user.js)
